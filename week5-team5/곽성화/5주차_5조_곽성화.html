[결론] 코로나19 방역 지침 완화 => 공연에 대한 수요 증가¶

  • 코로나19 발생 이후 공연예술 매출액 크게 떨어짐 -> 2020년 3~4월 최하점
  • 동반자 간 띄어 앉기(21-02-02 시행)가 자리 잡음 -> 21년 3월부터 안정적으로 운영
  • 사회적 거리두기 개편안(2021년 7월)으로 공연장 입장인원 최대 4,000명까지 상향
  • 방역패스를 중단 등 방역 지침 완화(2022년 3월) -> 지속적인 상승세
  • 코로나19 종식 이후 일상으로 돌아가기 위해 보복소비가 예상되므로 선제적 대응이 필요하다고 사료됨

# start_num ~ end_num까지 크롤링
def get_titles(start_num, end_num, search_word, title_lsit):
    
    while 1:
        if start_num > end_num:
            break
        url = 'https://search.naver.com/search.naver?where=news&sm=tab_jum&query={}&start={}'.format(search_word, start_num)
        req = requests.get(url)
        time.sleep(1)
        
        if req.ok:  # 정상적인 request 확인
            soup = BeautifulSoup(req.text, 'html.parser')
            
            # 뉴스제목 뽑아오기
            list_news = soup.find('ul', {'class' : 'list_news'})
            li_bxs = list_news.find_all('li', {'class': 'bx'})
            for li_bx in li_bxs:
                news_title = li_bx.find('a', {'class':'news_tit'})
                title_list.append(news_title['title'])
        start_num += 10
#     print(title_list)
